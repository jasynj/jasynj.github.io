The site showcases the company's services including catering, decoration, coordination, photography, videography, and MC/DJ management. SafeLink

SafeLink Mesh AI is a comprehensive offline-first mesh networking system designed for disaster relief and emergency communications. When traditional internet infrastructure fails, SafeLink enables peer-to-peer communication, emergency alerts, and relief coordination through Bluetooth Low Energy (BLE) and WiFi Direct mesh networks.

GSU Clubs & Organizations Portal

GSU Clubs Portal is a full-stack organization directory utilized by Grambling State University that centralizes every club and Greek organization in one place. Presidents can securely manage their org's constitution and bylaws documents through a password-protected editor, while admins maintain oversight across all organizations. It solves the problem of scattered or inaccessible org information by providing a clean public directory backed by a real database, with documents and file attachments persisted in cloud storage.

MayaCare

MayaCare is a maternal health advocacy platform built to address the Black maternal mortality crisis. It helps mothers track symptoms, prepare for doctor visits, and recognize warning signs using a transparent, rule-based triage system. By centering racial equity and self-advocacy, MayaCare equips mothers with tools to be heard, informed, and protected throughout pregnancy and postpartum care.

FitSync

FitSync is a web-based fitness tracking application that allows users to log meals and workouts, visualize their nutrition intake and calorie output, and stay consistent with fitness goals. It solves the problem of overwhelming or paywalled fitness tools by offering a clean platform powered by real food/exercise data via the Nutritionix API.

Demo Github Link

Reminisce

Reminisce is a voice-first AI memory companion built for seniors with early-stage memory loss, designed to feel like a caring companion rather than a traditional assistant. It uses a low-latency RAG architecture with long-term memory, natural speech interaction, and real-time visual reminders to help users remember personal details and daily tasks without any typing. Built with Gemini 2.0 Flash, Pinecone, FastAPI, and a custom ElevenLabs voice, Reminisce prioritizes speed, empathy, and accessibility.

Demo Github Link

QuantSim

QuantSim is an algorithmic trading backtesting platform for strategy research and performance analysis. It includes an event-driven backtesting engine with transaction costs, slippage, and latency modeling. Supports Momentum (MA Crossover), Mean Reversion (Z-Score), and Pairs Trading strategies.

Demo Github Link

Flixter

Flixster is a responsive movie discovery web application built with React and The Movie Database API. The app allows users to browse current movies in a dynamic grid layout, search by title, and sort results by release date, rating, or alphabetical order. Each movie card displays key details and opens a modal with extended information, including runtime, genres, overview, and backdrop imagery. The project emphasizes accessibility, semantic HTML, responsive design, and clean user interactions while demonstrating real-world API integration, state management, and component-driven UI development.

Demo Github Link

VisionCare

VisionCare is a lightweight web application inspired by the EyecareLive Vision Test, designed to help users assess their visual acuity from home using a laptop or desktop paired with a smartphone. The application guides users through the testing process and allows them to securely save and view their test results. It’s intended as a preliminary screening tool, NOT a replacement for a full eye exam.
